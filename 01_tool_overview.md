# PAL Vocabulary Support Tool — MVP Overview

## Purpose
A lightweight, web-based supplemental learning tool for Class 3 students in Ahmedabad, Gujarat, India.
The tool supports students who struggle with in-class English instruction by reinforcing vocabulary,
decoding, reading, and writing skills at home using a tablet and slow internet.

## Target Audience
- Primary: Students aged 8–10 (Class 3)
- Secondary: English teachers using PAL-style remediation
- Context: Non–first-language English learners

## Stakeholders
- Students
- Teachers
- School administrators (indirect)
- Curriculum designers

## Core Principles
- No punishment, no correction shown
- Diagnostic, not instructional replacement
- Encourages confidence and exposure
- Designed for low bandwidth and basic hardware

## Key Constraints
- No student audio recording
- Very slow or unreliable internet
- No passwords for students
- Teachers control lesson access

## Primary Use Case
1. Teacher teaches concept in class
2. Teacher identifies struggling students
3. Student takes tablet home
4. Student completes unlocked lesson(s)
5. Results sync automatically
6. Teacher reviews progress next day

## Non-goals (MVP)
- Pronunciation scoring
- Parent dashboards
- Adaptive tutoring or hints
