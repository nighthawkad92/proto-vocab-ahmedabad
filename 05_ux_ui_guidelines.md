# UX / UI & Tone of Voice Guidelines

## Audience Fit
- Age: 8–10
- Reading level: basic
- Emotional goal: confidence, curiosity, calm

## Visual Style
- Large rounded buttons
- Bright but not neon colours
- Friendly mascot presence (optional)
- Minimal text per screen

## Design System
- Tailwind CSS
- Headless UI
- Framer Motion (light use)
- Lottie animations

## Interaction Patterns
- Tap > response
- No long scrolls
- Gentle transitions
- Clear state changes (idle, active, success)

## Animation
- Entry: soft slide/fade
- Feedback: bounce or glow
- Loading: playful loop (2–3s max)

## Sound Design
- Soft pop / chime for taps
- Encouraging sound on completion
- No harsh error sounds

## Tone of Voice
- Curious: “Let’s try something fun.”
- Encouraging: “Nice thinking!”
- Light humour: “That word is a long one!”
- Persona: Enthusiastic linguist, simple words, warm energy

## Accessibility
- High contrast
- Audio for all instructions
- Touch-only usable
