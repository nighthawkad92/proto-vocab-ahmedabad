# EXECUTION SPEC — Accessibility & Inclusion Layer

## Role for Claude
Ensure usability on low-end tablets.

## Touch Rules
- Target size ≥ 48px
- Spacing ≥ 8px

## Visual Rules
- Contrast ≥ WCAG AA
- Never rely on color alone

## Audio Rules
- All audio has visual cue
- All instructions available as text

## DO NOT
- Use small icons
- Hide critical actions
