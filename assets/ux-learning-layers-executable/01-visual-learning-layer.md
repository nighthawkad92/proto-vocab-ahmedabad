# EXECUTION SPEC — Visual Learning Layer

## Role for Claude
Act as a senior UI designer for a children's ESL vocabulary app (age 8–10, Ahmedabad, India).
Optimize for clarity, low cognitive load, and low-spec tablets.

## Allowed Resources
- Icon libraries: Material Icons, Heroicons (SVG only)
- Illustration sources: Simple SVG shapes only (no complex illustrations)
- Fonts: System fonts or Google Fonts (e.g. Noto Sans, Inter)

## Layout Rules
- One learning concept per screen
- Max 2 primary visual elements per screen (word + object)
- Use vertical stacking, no side-by-side complexity
- Touch targets ≥ 48px

## Color & Typography
- High contrast (WCAG AA minimum)
- Background: neutral light color
- Primary action: single accent color
- Font sizes:
  - Word: 28–32px
  - Instruction: 16–18px

## Visual Explanation Pattern
1. Show object/image
2. Show word label beneath
3. Highlight relationship (outline, arrow, or fade-in)

## DO NOT
- Add decorative imagery
- Use gradients, shadows, or glass effects
- Introduce new layouts mid-lesson
